Descripción del Proyecto

Permite al iniciar sesión como administrador, administrar usuarios jugadores de valorant (ascender, eliminar...), crear partidas entre los jugadores, administrar las características de esta y modificarlas.
Al iniciar sesión como jugador o registrarse, permite ver sus datos y colección (armas favoritas, mapas...) y modificarlos. También tienen la posibilidad de ver las partidas que tiene que jugar y los datos de las mismas.

Características Principales

  • Interfaz intuitiva y fácil de usar
  • Validación y control de excepciones tratadas en la interfaz de usuario
  • Conexión con base de datos a tiempo real
  • Modelo complejo de base de datos relacional
  • Operaciones CRUD

Tecnologías Utilizadas

Este proyecto está construido con Java, conectando con una base de datos en MySQL. Con librerías como JavaSwing para poder crear ventanas para mejorar la experiencia del usuario.
